# Summary

Update: Added a Version 2 in order to strengthen the fine detail extended tower as it might fail depending on the filament type

This is a Remix of the Ultimaker tower which i modified slightly to also include bridging, stringing and also fine detail. No gcode supplied. The temperatures need to be either changed manually or of you are using Simplify3D you can simply set mulitple layers for the hotend temperatures.
